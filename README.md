# MusicCollab Platform

A full-stack web application for musicians to collaborate.

## Features
- User Authentication (Login/Register)
- Create and view posts
- Protected routes

## Tech Stack
- Frontend: React (Vite)
- Backend: Spring Boot

## Live Demo
https://music-collab-platform-08yw.onrender.com

## Run Locally

### Frontend
cd frontend  
npm install  
npm run dev  

### Backend
Run Spring Boot app
